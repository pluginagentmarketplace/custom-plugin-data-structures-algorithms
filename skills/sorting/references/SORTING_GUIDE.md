# Sorting Guide

| Algorithm | Time | Space | Stable |
|-----------|------|-------|--------|
| QuickSort | O(n log n) | O(log n) | No |
| MergeSort | O(n log n) | O(n) | Yes |
| HeapSort | O(n log n) | O(1) | No |
| CountingSort | O(n+k) | O(k) | Yes |

## When to Use
- **QuickSort**: General purpose, in-place
- **MergeSort**: Stability needed, linked lists
- **HeapSort**: Memory constrained
