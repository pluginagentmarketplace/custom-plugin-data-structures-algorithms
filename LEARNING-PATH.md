## ðŸŸ¡ Intermediate Path (2-3 weeks)

**For**: Those with DSA basics, preparing for interviews
**Goal**: Solve medium LeetCode problems (75+)
**Time**: 30-40 hours total
**Difficulty**: 20 easy, 80 medium, 10 hard problems

### Week 1: Trees & Graphs Deep Dive (Days 1-7)

**Agents**: Tree Master + Graph Guide
- Days 1-3: Advanced tree problems (3 hours)
  - Complex tree traversals
  - Binary search trees edge cases
  - Tree serialization

- Days 4-7: Graph algorithms (4 hours)
  - Shortest path algorithms
  - MST algorithms
  - Topological sort

**Skills**: Tree Structures + Graph Algorithms
**Practice**: 30+ medium problems (15 trees, 15 graphs)
**Time**: 7 hours

### Week 2: Dynamic Programming (Days 8-14)

**Agent**: Dynamic Programming Strategist
- Days 8-10: DP foundations (3 hours)
  - State definition
  - Recurrence relations
  - Memoization vs tabulation

- Days 11-14: DP patterns (4 hours)
  - 1D DP problems
  - 2D DP problems (knapsack)
  - String DP

**Skill**: Dynamic Programming
**Practice**: 25+ DP problems
**Time**: 7 hours

### Week 3: Interview Prep & Mock (Days 15-21)

**Agents**: All agents for weaknesses + Advanced Techniques
- Days 15-18: Identify weak areas (2 hours)
  - Take `/assess` intermediate test
  - Review weak topics

- Days 19-21: Mock interviews (5 hours)
  - Take 2-3 mock interviews
  - Get feedback
  - Optimize solutions

**Commands**: `/interview-prep` for mocks
**Practice**: 15+ problems on weak areas
**Time**: 7 hours

**Progression**:
- Week 1: Master trees and graphs
- Week 2: Master dynamic programming
- Week 3: Interview ready for medium problems
- **Total**: 80-100 medium problems solved
