# /learn

Start your Data Structures & Algorithms learning journey with a structured path.

## Description

This command guides you through selecting a learning path based on your current skill level and goals.

## How to Use

Simply type `/learn` to begin. You'll be asked:

1. **Your Current Level**
   - Beginner: New to DSA concepts
   - Intermediate: Familiar with basic data structures
   - Advanced: Ready for complex algorithms

2. **Your Goal**
   - Interview Preparation: Optimize for coding interviews
   - Competitive Programming: Speed and efficiency
   - System Design: Real-world applications
   - Comprehensive Mastery: Deep understanding

3. **Time Commitment**
   - Quick Path (2-3 weeks)
   - Standard Path (6-8 weeks)
   - Deep Dive (12+ weeks)

## What You'll Get

- Personalized learning sequence
- Recommended agents for your path
- Practice problems targeted to your level
- Interview tips and strategies
- Project suggestions

## Example Paths

### Interview Preparation Path (8 weeks)
- Week 1-2: Foundations & Linear Structures
- Week 3: Searching & Sorting
- Week 4: Non-Linear Structures
- Week 5: Dynamic Programming
- Week 6: Graph Algorithms
- Week 7: Advanced Topics
- Week 8: Mock interviews & practice

### Competitive Programming Path (6 weeks)
- Week 1: Foundations & Bit Manipulation
- Week 2-3: Advanced Data Structures
- Week 4: Graph Algorithms & Optimization
- Week 5: Dynamic Programming
- Week 6: Competition practice

## Tips

- Commit to consistent daily practice
- Code every solution yourself
- Understand before optimizing
- Revisit difficult concepts
- Join online judge platforms

## Next Steps

After selecting your path:
1. Review agent descriptions with `/explore-agent`
2. Start with recommended agent
3. Use `/practice` for problems
4. Track your progress
