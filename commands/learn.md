# /learn - Start Your Learning Journey

Begin your Data Structures & Algorithms learning path with this command.

## Usage

```
/learn
```

## What This Command Does

This command will:

1. **Assess Your Current Level**
   - Beginner: Never studied DSA or just starting
   - Intermediate: Know basics, want to deepen knowledge
   - Advanced: Solid foundation, preparing for interviews

2. **Create Your Learning Path**
   - Personalized recommendations based on level
   - Optimal progression through topics
   - Estimated time commitment

3. **Connect You to Resources**
   - Specific agent recommendations
   - Relevant skill modules
   - Practice problems suited to your level

## Getting Started

After running `/learn`:

1. Select your skill level
2. Review your personalized learning path
3. Start with the recommended agent
4. Access the associated skill modules
5. Practice with provided problems

## Commands to Use After

- `/browse-agent` - Explore specific agents
- `/assess` - Test your knowledge
- `/interview-prep` - Practice interviews
- `/projects` - Build hands-on projects

## Tips for Success

- **Consistency**: 1-2 hours daily beats cramming
- **Practice**: 70% coding, 30% theory
- **Understand**: Don't memorize; understand patterns
- **Teach**: Explain solutions to someone else
- **Track Progress**: Mark completed problems
