# Learn - Start Your Developer Journey

Welcome to the Developer Roadmap Plugin! This command helps you select a learning path based on your interests and current experience level.

## Available Roles

### 1. Backend Developer
Build scalable APIs, databases, and server-side applications.
- **Roadmap**: https://roadmap.sh/backend
- **Skills Focus**: Node.js, Python, Java, APIs, Databases
- **Duration**: 6-12 months

### 2. Frontend Developer
Create beautiful, responsive web applications.
- **Roadmap**: https://roadmap.sh/frontend
- **Skills Focus**: React, Vue, Angular, JavaScript, CSS
- **Duration**: 6-12 months

### 3. DevOps Engineer
Manage infrastructure, containers, and CI/CD pipelines.
- **Roadmap**: https://roadmap.sh/devops
- **Skills Focus**: Docker, Kubernetes, CI/CD, AWS
- **Duration**: 8-14 months

### 4. Data Scientist & AI Engineer
Build machine learning models and data-driven solutions.
- **Roadmap**: https://roadmap.sh/ai-engineer
- **Skills Focus**: Python, ML, Deep Learning, TensorFlow
- **Duration**: 8-14 months

### 5. System Architect
Design large-scale distributed systems.
- **Roadmap**: https://roadmap.sh/system-design
- **Skills Focus**: System Design, Scalability, Patterns
- **Duration**: 10-16 months

### 6. Mobile Developer
Build iOS, Android, and cross-platform applications.
- **Roadmap**: https://roadmap.sh/ios / https://roadmap.sh/android
- **Skills Focus**: Swift, Kotlin, React Native, Flutter
- **Duration**: 8-14 months

### 7. Cloud Engineer
Master AWS, Azure, and cloud infrastructure.
- **Roadmap**: https://roadmap.sh/aws
- **Skills Focus**: AWS, Cloud Architecture, Infrastructure
- **Duration**: 6-12 months

## Learning Paths by Level

### Beginner (0-3 months)
- Programming fundamentals
- Version control with Git
- Basic web development
- Database basics

### Intermediate (3-6 months)
- Framework mastery
- API development
- Testing strategies
- Deployment basics

### Advanced (6-12 months)
- System design
- Performance optimization
- Architecture patterns
- Production deployment

## Getting Started

1. Choose your role above
2. Visit the roadmap link
3. Follow the learning path
4. Use `/assess` to test your knowledge
5. Use `/projects` for hands-on practice

## Additional Resources

- Use `/browse-agent` to explore all agents
- Use `/assess` for knowledge testing
- Use `/projects` to find practice projects
- Check out skill documentation for quick references
