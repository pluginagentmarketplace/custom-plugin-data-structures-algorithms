## Practice Tips

1. **Read problem carefully** - Clarify constraints and edge cases
2. **Explain approach** - State the algorithm before coding
3. **Code cleanly** - Use descriptive variable names
4. **Test thoroughly** - Check edge cases
5. **Optimize** - Time and space complexity
